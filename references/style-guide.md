# 风格指南 · 水彩速写（Watercolor Reportage Sketch）

风格样本：`assets/example-watercolor-sketch.jpeg`（新闻现场速写，证词场景）。风格源自新闻现场速写画师的视觉语言——墨水勾线 + 水彩薄涂记录现场，**题材不限于法庭**：会议、街头、办公室、发布会、人物特写、事件现场均适用。

## 一、风格要素拆解

| 维度 | 特征 |
|------|------|
| 媒介 | 钢笔/针管笔墨水勾线 + 水彩湿画法薄涂，纸本 |
| 线条 | 松动、快速、手绘速写感；轮廓线不完全闭合；有重复试探线 |
| 色彩 | 透明水彩罩染，色层薄、可见纸纹；边缘自然洇开 |
| 色板 | 淡黄、粉紫、薰衣草紫、天蓝、木色棕、暖粉肤色、灰绿点缀 |
| 背景 | 大面积单色晕染 wash + 大量留白，不刻画细节 |
| 人物 | 比例略带速写夸张，面部表情生动，动态抓拍感 |
| 构图 | 旁观者/现场记者视角，中远景群像或近景人物，有前后纵深 |
| 细节 | 手写体场景标注（**不含画师签名、不含日期时间**），新闻纪实质感 |
| 画幅 | 横版 16:9（公众号头图），也可 3:2 |
| 题材 | 不限：庭审、商务、科技、财经、社会、文化、人物、事件现场 |

## 二、风格锚定段（逐字复用，勿改写）

### 英文锚定段（作为主 prompt 前缀）

```
Reportage sketch illustration, traditional news sketch artist's hand-drawn ink
and watercolor on paper. Loose black ink pen outlines with visible
hand-drawn sketchy strokes, contour lines not fully closed, occasional searching
repeat lines. Transparent watercolor washes layered over the ink, thin color layers
with visible paper texture and soft bleeding edges. Pastel palette: pale yellow,
dusty pink, lavender purple, sky blue, warm wood brown, rosy skin tones. Background
rendered as a soft monochrome watercolor wash with generous white space. Figures
slightly exaggerated in a lively reportage manner, expressive faces and captured
gestures, on-the-spot documentary journalism feel. Handwritten-style scene
annotations, but no readable artist signature and no dates or timestamps anywhere
in the image. Horizontal 16:9 composition, observer's viewpoint.
```

### 中文锚定段（供中文出图工具或人工核对）

```
新闻速写水彩插画，传统新闻速写画师的纸本钢笔水彩手绘。黑色墨水钢笔松动勾线，
速写笔触明显，轮廓线不完全闭合，偶有试探性重复线。墨线上罩透明水彩薄涂，
色层轻薄可见纸纹，边缘自然洇开。粉彩色板：淡黄、灰粉、薰衣草紫、天蓝、
暖木棕、玫瑰肤色。背景为单色水彩大面积晕染并大量留白。人物略带速写式夸张，
表情生动、动作有抓拍感，现场新闻纪实气质。画面可有手写体场景标注，
但不出现任何画师签名、日期或时间。
横版 16:9，旁观者视角。
```

### 负面约束（每次必带）

```
no photorealism, no photography, no 3D render, no CGI, no digital smooth gradients,
no anime or manga style, no oil painting, no thick impasto, no neon colors,
no sharp vector lines, no flat cartoon fill,
no artist signature, no readable signature, no dates, no timestamps, no watermark
```

## 三、场景描述写法

锚定段之后接场景段，只写变量：人物 + 动作 + 环境 + 氛围，英文 1–3 句。**任何题材都可以**，只要写成"现场的人在做具体的事"。

好例子（覆盖不同题材）：

```
Scene: a founder on a launch-event stage presenting a glowing new device to a
crowd of journalists holding up phones and notebooks, spotlit, expectant atmosphere.
```

```
Scene: two traders arguing across a desk stacked with paper reports and a glowing
stock-ticker screen, late evening office, exhausted but defiant mood.
```

```
Scene: an elderly craftsman repairing a watch at a cluttered workbench in a small
street-side shop, a young customer leaning in to watch, quiet concentrated mood.
```

避免：罗列风格词（风格已在锚定段）、抽象概念（"内卷""焦虑"要转译为具体的人和动作）。

## 四、抽象概念 → 场景转译速查

| 文章主题 | 场景化写法示例 |
|----------|----------------|
| 庭审/法律事件 | 证人席、律师陈词、旁听席反应、证据展示屏 |
| 公司/商业新闻 | 会议室对峙、发布会讲台、深夜办公室、签约握手 |
| 财经/市场 | 交易屏前的交易员、公告栏前围观人群、摔落的文件 |
| 科技/产品 | 发布会舞台、工程师围在白板前、实验室操作台 |
| 社会/纪实 | 街头采访、排队人群、当事人背影与记者话筒 |
| 文化/生活 | 市集摊位、咖啡馆长谈、书店角落阅读的人 |
| 人物特写稿 | 当事人侧脸近景 + 背景晕染，手边标志性物件 |
| 数据/观点 | 比喻场景：天平、空椅子、长桌两端的人 |

## 五、公众号排版建议

- 头图：16:9 横版，选全文冲突最强或最有画面感的场景。
- 文中插图：每 800–1200 字一张，与相邻段落内容对应。
- 同一篇 2–4 张为宜，全部共用同一锚定段与画幅，保证翻阅时风格连贯。
