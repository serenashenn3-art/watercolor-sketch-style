# Watercolor Sketch Style · 水彩速写配图

为**任何题材**的文章或简短需求生成统一风格的水彩速写新闻纪实插画。核心资产：固定「风格锚定段」，保证同一篇文章所有插图风格统一。

## 风格定义（一眼识别）

参照 `assets/example-watercolor-sketch.jpeg`：

- 黑色墨水钢笔勾线，线条松动、速写感强、不完全闭合
- 透明水彩薄涂罩染，色层可见纸纹、边缘自然洇开
- 粉彩色板：淡黄、粉紫、天蓝、木色棕、暖粉肤色
- 背景大面积水彩晕染 + 留白
- 人物略带速写夸张、表情生动，现场抓拍感，旁观者视角
- 手写体场景标注（**不出现画师签名、日期、时间**），新闻纪实质感
- 横版 16:9（公众号头图），绝不照片写实
- **题材不限**：法庭、会议、街头、发布会、办公室、人物特写皆可入画

完整风格拆解与提示词模板见 `references/style-guide.md`。**组装提示词前必读该文件中的锚定段原文，不要凭记忆改写。**

## 工作流

### 模式 A：文章配图（输入为任意文章）

1. 通读文章，选出 2–4 个「关键内容场景」。优先级：冲突/转折时刻 > 核心人物动作 > 关键关系或场景氛围 > 抽象概念的场景化比喻。每场景用一句中文描述：谁、在哪、做什么、什么情绪。抽象主题（观点、数据、趋势）按 style-guide.md 第四节转译为具体的人和动作。
2. 为每个场景组装提示词 = 固定风格锚定段 + 场景描述（英文）+ 画幅参数。用脚本组装：
   ```bash
   python scripts/build_prompt.py --scene "<场景中文描述>"
   ```
   脚本输出中英双语完整提示词。
3. 出图（见下方「出图与降级」），同一篇文章的所有插图使用**同一锚定段、同一画幅**，逐张生成。
4. 交付：图片文件（或提示词清单）+ 每张图对应的文章段落说明，建议插入位置。

### 模式 B：简短需求（输入为一句话）

直接用脚本组装单条提示词并出图：
```bash
python scripts/build_prompt.py --scene "深夜办公室里，两名交易员隔着堆满报表的桌子争论"
```

## 出图与降级

1. 检查当前环境可用的图像生成能力，任选其一：用户指定的 **GPT image2 / gpt-image 接口**、**nano banana（Gemini 图像生成）**、Kimi 的 **image_generation 插件**、即梦、Midjourney 等。用户指定了工具就优先用用户指定的。
2. 可用 → 逐张调用生成，画幅 16:9（公众号头图）或按用户要求；把英文提示词作为主 prompt。
3. 不可用 → 明确告知"当前环境没有可调用的图像生成工具"，输出**完整详细的中英双语提示词清单**（锚定段 + 场景段 + 负面约束，逐条齐全），方便用户自行修改后粘贴到任何出图工具使用。
4. 生成后检查：是否墨水勾线 + 水彩薄涂、粉彩色板、现场纪实感、无照片写实感。跑偏时收紧负面约束重生成（负面清单见 style-guide.md）。

## 实测经验（调用 image_generation 插件时）

- **prompt 里避免写真实画师姓名**，会触发版权/内容审核（HTTP 403）。统一使用通用风格描述（"traditional news sketch artist's hand-drawn ink and watercolor"），风格锁定的效果一样。
- **生成图中不得出现画师签名、日期、时间**：锚定段与负面约束均已内置该禁令；参考图本身带有签名和日期，生成后必须目检，若模型模仿参考图带出了签名/日期，收紧负面约束重生成。
- **风格一致性靠参考图**：先用插件的 `image-to-url` 把 `assets/example-watercolor-sketch.jpeg` 转成公共 URL，每张生成时以 `--reference-image` 传入，整套图风格即高度统一（含手写场景标注质感）。
- 带参考图若返回 HTTP 424（服务暂时不可用），去掉参考图重试即可，通用锚定段单独使用也有不错效果。
- 2K + 16:9（2048×1152）适合公众号头图，已验证可用。

## 风格统一的三条铁律

1. 锚定段逐字复用，每次生成都带完整锚定段，不缩写。
2. 同一批插图共用同一色板与画幅；场景描述只换人物/动作/环境。
3. 负面约束每次必带：no photorealism, no 3D render, no digital smooth gradient, no anime, no oil painting, no artist signature, no dates。
